Python MCQ Quiz

1. What is the data type of `42` in Python?

Correct Answer: b) int
The value `42` is a whole number, so its data type in Python is `int`. `float` is for decimals, `str` for strings, and `bool` for True/False.

2. What is the output of `type("Python")`?

Correct Answer: c) str
The `type()` function returns the data type. `"Python"` is a string, so its type is `str`. Other options represent different data types.

3. How do you create a variable named `score` with the value 100?

Correct Answer: b) score = 100
In Python, variables are created by assigning a value using `=`, e.g., `score = 100`. No keyword or type declaration is needed. Other options are syntax from languages like JavaScript, C, or Go.

4. What is the data type of `False` in Python?

Correct Answer: d) bool
`False` is a boolean value in Python, so its type is `bool`. It represents a logical value (True/False). Other options are incorrect for this value.

5. What is the output of `float(5)`?

Correct Answer: b) 5.0
The `float()` function converts a number to a floating-point value. `float(5)` converts the integer `5` to the float `5.0`.

6. What is the output of `7 + 2 * 3`?

Correct Answer: b) 13
Python follows PEMDAS (order of operations). Multiplication (`2 * 3 = 6`) is evaluated first, then addition (`7 + 6 = 13`). Use parentheses for a different order, e.g., `(7 + 2) * 3`.

7. Which operator checks if two values are not equal?

Correct Answer: b) !=
The `!=` operator checks if two values are not equal. `==` checks equality, `=` is for assignment, and `>=` and `<=` are for comparison.

8. What is the output of `10 - 6`?

Correct Answer: a) 4
The `-` operator subtracts the second number from the first. `10 - 6` equals `4`.

9. What is the output of `"Hello"[0]`?

Correct Answer: a) H
String indexing in Python starts at 0. `"Hello"[0]` accesses the first character, which is `H`.

10. What is the output of `9 * 3`?

Correct Answer: a) 27
The `*` operator multiplies two numbers. `9 * 3` equals `27`.

11. What is the output of `"code"[2]`?

Correct Answer: c) d
String indexing in Python starts at 0. For `"code"`, index 2 corresponds to the third character, which is `d` (c=0, o=1, d=2).

12. What is the output of `"TEST".lower()`?

Correct Answer: b) test
The `lower()` method converts a string to lowercase. `"TEST"` becomes `"test"`.

13. What is the output of `"py" + "thon"`?

Correct Answer: b) python
The `+` operator concatenates strings in Python. `"py" + "thon"` combines them into `"python"`. No spaces are added automatically.

14. How many characters are in the string `"hello"`?

Correct Answer: b) 5
The `len()` function returns the number of characters in a string. For `"hello"`, there are 5 characters: h, e, l, l, o.

15. What is the output of `str(123)`?

Correct Answer: b) "123"
The `str()` function converts a number to a string. `str(123)` converts the integer `123` to the string `"123"`.

16. What is the output of `15 % 4`?

Correct Answer: a) 3
The `%` operator returns the remainder of division. `15 % 4` means 15 divided by 4 is 3 with a remainder of 3 (since 4 * 3 = 12, and 15 - 12 = 3).

17. How many times does `for i in range(3):` loop?

Correct Answer: b) 3
The `range(3)` generates numbers from 0 to 2 (0, 1, 2), so the loop runs 3 times.

18. What is the output of `"Fun"[1]`?

Correct Answer: b) u
String indexing in Python starts at 0. For `"Fun"`, index 1 corresponds to the second character, which is `u` (F=0, u=1).

19. What is the output of `8 > 5`?

Correct Answer: a) True
The `>` operator checks if the first value is greater than the second. Since `8` is greater than `5`, the result is `True`.

20. What does `if x >= 10:` do?

Correct Answer: a) Executes block if x is 10 or greater
The `if` statement with `>=` checks if `x` is greater than or equal to 10. If true, the block executes; otherwise, it skips the block.

21. What is the output of `True or False`?

Correct Answer: a) True
The `or` operator returns `True` if at least one operand is `True`. Since the first operand is `True`, the result is `True`.

22. What does `if not True:` do?

Correct Answer: b) Skips block
The `not` operator inverts a boolean. `not True` is `False`, so the block under `if not True:` will not execute.

23. What is the output of `"DATA".upper()`?

Correct Answer: b) DATA
The `upper()` method converts a string to uppercase. Since `"DATA"` is already uppercase, it remains `"DATA"`.

24. What does `elif x == 5:` do in an if statement?

Correct Answer: a) Checks x == 5 if the previous condition was false
The `elif` statement is used in an if-else structure. It checks its condition (`x == 5`) only if the previous `if` or `elif` condition was false.

25. What is the output of `3 < 7`?

Correct Answer: a) True
The `<` operator checks if the first value is less than the second. Since `3` is less than `7`, the result is `True`.

26. What is the data type of `7.5` in Python?

Correct Answer: b) float
The value `7.5` is a decimal number, so its data type in Python is `float`. `int` is for whole numbers, `str` for strings, and `bool` for True/False.

27. What does `for i in range(5):` do?

Correct Answer: a) Loops 5 times, with i from 0 to 4
The `range(5)` generates numbers from 0 to 4 (0, 1, 2, 3, 4), so the loop runs 5 times, with `i` taking each value in sequence.

28. What is the output of `12 / 3`?

Correct Answer: a) 4.0
The `/` operator performs division and returns a float in Python 3. `12 / 3` equals `4.0`.

29. What is the value of `i` in the first iteration of `for i in range(2):`?

Correct Answer: a) 0
The `range(2)` generates numbers from 0 to 1 (0, 1). In the first iteration, `i` takes the value `0`.

30. What is the output of `int("45")`?

Correct Answer: a) 45
The `int()` function converts a string representing a number to an integer. `"45"` becomes the integer `45`.
